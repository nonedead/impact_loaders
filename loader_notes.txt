Series 6  Mediatechnics Impact Loader Code notes for the Impact loader 89C51 processor  3/13/15

The TurnKey folder contains the original code and assembler as received from R-Quest as well as TurnKey Solutions-generated code that adapts the Series 6 loader commands to the Impact Loader command set.   There are 2 parts to the Impact loader code, the boot code and command/control code.

The Boot folder is Company Loader Boot
A51BOOT.A51  12K  7/22/2002  9:32 A.M. is the original R-Quest code written for Series 4 duplicator remote firmware updates for the Mediatechnics Impact loader.  The Series 6 duplicator doesn't support remote firmware programming for Impact loaders.  The A51Boot code was not modified to accommodate Series 6 loader control.  
Use BTImpact.hex for programming the Impact 89C51 processor.  Load this code @ 0000H.  


The 104P folder contains code for Series 6 duplicator control of Impact loaders.  
Impact.A51   59K  7/26/2012   5:07 P.M. is the TurnKey Solutions-generated code to assemble into a .hex file.  
Impact.Hex   46K  7/26/2012  5:09 P.M. is the  loader code for loading into the processor @ 4000H

ImpactR2.txt  58K  10/4/2010  4:50 P.M. is the original R-Quest code written for impact loaders controlled by Series 4 duplicators.  This code was used to produce Impact.A51.  

Compatible processors for the Mediatechnics Impact loader are:
Philips 89C51RC+IN  40 pin DIP  32K memory.  This processor has the highest immunity to motor noise.  
Philips 89C51RC2BN 40 pin DIP  32K memory
Do not use the Philips 89C51RD2BN processor.  It has a poor immunity to motor noise generated by the loader.  

Programming the 89C51 is accomplished via a Xeltek Superpro Model M, running Superpro V 1.0 Sofware D/C: 10/21/2011.
